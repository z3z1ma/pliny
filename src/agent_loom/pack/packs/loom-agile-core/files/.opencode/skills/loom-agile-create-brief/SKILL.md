## Brief outline
- Problem
- Target users
- Desired outcomes (measurable)
- Non-goals
- Constraints
- Risks / unknowns
- Next step recommendation

## Procedure
1) Confirm ticket and intent
2) Ask only the questions required to avoid guessing
3) Draft the brief in bullets
4) Add a ticket note summarizing the brief and where it lives
