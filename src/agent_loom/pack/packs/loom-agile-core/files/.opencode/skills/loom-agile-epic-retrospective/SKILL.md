## Prompts
- What went well?
- What was slow or confusing?
- What should we change next time?
- What should become a reusable skill?

## Procedure
1) Gather signals (ticket notes, incidents, delays)
2) Write a short retro summary
3) Persist:
   - ticket note
   - 1-3 Loom memory notes for durable learnings
