## Procedure
1) Confirm objective
2) Inventory candidate stories/tickets
3) Capacity + constraints
4) Select scope
5) Sequence work and identify dependencies
6) Define done criteria
7) Update tickets
   - set selected tickets to `ready` or `in_progress`
   - add notes for sequencing and dependencies
