# Ledger Import Terms

## Glossary Terms

- `sourceRef`: settled name for the source-row identifier emitted in Ledger
  import preview output.
- `externalId`: rejected older name for the same concept. Do not use it in new
  Ledger import tests.

## Conventions And Preferences

Ledger import preview tests should keep row-source naming consistent with the
product contract and existing review language.
